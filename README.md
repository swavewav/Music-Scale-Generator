# Music-Scale-Generator 

A **front-end music tool** that allows users to explore musical scales and play piano notes directly in the browser. Built with **HTML, CSS, and JavaScript**, this app demonstrates **scale generation, DOM manipulation, and interactive audio playback**.

---

## Portfolio Pitch

This project showcases skills in **front-end interactivity, musical theory logic, and audio handling**. Users can select a root note and scale type to generate scales dynamically, and interact with a playable piano keyboard.

---

## Features

- Generate **major, minor, pentatonic, blues, harmonic minor, dorian, and mixolydian scales**.
- Dynamic scale display based on **root note** and **scale type**.
- Clickable **piano keyboard** to play notes.
- Keyboard shortcuts to play notes as if using a real piano.
- Adjustable **volume slider** for playback control.
- Responsive design optimized for mobile and desktop.

---

## Limitations

- The piano sounds are **static audio files** and are **not connected to a live MIDI interface**.
- The scales are **display-only**; playing the piano does not adapt to the selected scale.
- No recording, saving, or exporting of scales or piano sessions.

---

## Potential Upgrades

- Link piano playback to the **currently selected scale**, highlighting only valid notes.
- Add a **visual note highlight** on scale generation to guide learning.
- Integrate **MIDI support** for advanced keyboard control.
- Allow users to **record, save, or export** scales and performances.
- Implement **different instrument sounds** besides piano.

---

## How to Run

1. Clone or download the repository.
2. Ensure the **sounds folder** containing the piano `.wav` files is in the project root.
3. Open `index.html` in a modern web browser.
4. Use the **dropdown menus** to select a root note and scale type.
5. Click the **piano keys** or press keyboard keys to play notes.

---
